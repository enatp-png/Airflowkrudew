# Air Flow Mission Group v4

เพิ่มตามคำขอ:
1. ส่งงานเป็นกลุ่ม เลือกกลุ่มแล้วเห็นสมาชิกทุกคนทันที
2. ด่าน 3 เป็นแผนที่ประเทศไทย + ตัวอย่างลมตะวันตกเฉียงใต้
3. ด่าน 4 เป็นแผนที่ประเทศไทย + สัญลักษณ์ SW/NE ให้นักเรียนลากเส้น
4. Admin แนบภาพโจทย์ด่าน 5 แยกแต่ละกลุ่มได้
5. ผลงานแยกเป็นแกลเลอรีรายกลุ่ม
6. มี gallery.html ให้นักเรียนดูผลประเมินและกดนำเสนอหน้าชั้นเรียน

วิธีอัปเดต:
1. รัน supabase.sql ใน Supabase SQL Editor
2. อัปโหลด index.html, admin.html, gallery.html, config.js, supabase.sql ขึ้น GitHub
3. กด Ctrl + Shift + R
