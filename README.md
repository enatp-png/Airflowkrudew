# Air Flow Mission Cute v3

เวอร์ชันนี้แก้ตามคำขอ:
- ปรับฟอนต์เป็น Noto Sans Thai ให้อ่านง่าย น่ารัก เป็นระเบียบ
- เพิ่มปุ่ม “เข้าหน้าครู / เพิ่มนักเรียน” บนหน้าแรก
- หน้า Admin มีปุ่มกลับหน้ากิจกรรม
- นักเรียนเลือกกลุ่มแล้วระบบแสดงรายชื่อในกลุ่ม
- ครูเพิ่มกลุ่ม / เพิ่มนักเรียน / นำเข้ารายชื่อหลายคน ได้ที่ `admin.html`

## วิธีอัปเดต
1. รัน `supabase.sql` ใน Supabase SQL Editor
2. อัปโหลดไฟล์ทั้งหมดขึ้น GitHub ทับของเดิม
3. ตรวจให้ชื่อไฟล์เป็น:
   - `index.html`
   - `admin.html`
   - `config.js`
   - `supabase.sql`
4. เปิดหน้าเว็บใหม่แล้วกด Ctrl + Shift + R

## รหัสครู
ค่าเริ่มต้นคือ `1234` แก้ได้ใน `config.js`
